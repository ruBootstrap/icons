Bootstrap Icons

Впервые в истории Bootstrap есть собственная библиотека иконок, специально разработанная и созданная для наших компонентов и документации - и теперь доступная для любого проекта.

Иконки Bootstrap предназначены для работы с компонентами Bootstrap, от элементов управления формы до навигации. Но они также будут работать практически в любом проекте. Это связано с тем, что иконки Bootstrap - это SVG, поэтому они быстро и легко масштабируются и могут быть стилизованы с помощью CSS.

Они имеют открытый исходный код (MIT), поэтому Вы можете бесплатно загружать, использовать и расширять любой значок. Каждый SVG можно скопировать прямо здесь, или Вы можете использовать SVG спрайт.
